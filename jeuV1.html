GOOD TO KNOW :

The CSS position property allows you to organize elements on the screen.
Position can be : static, relative, absolute, fixed.
When setting the position, you can use the following properties:
top, right, bottom and left.

Static

The default value of an element is static. It means that the element is not positioned in any particular way.
Offsets don't work on it ;)

Relative

You can adjust the position by setting the top, left, right, and bottom properties.
It will move the element relative to where it would normally occur in the document. Other content will not be adjusted to fit into any gap left by the element.

Absolute

The element is positioned to the nearest positioned element in the DOM( with a position other than static.)

Fixed

The element will always be in the same place, even when you scroll.
A fixed element is positioned relatively to the screen of your device (viewport).
